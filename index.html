Color Coding System

Throughout these materials, we use a consistent color coding system:

  • Red: Vocabulary words and phrases
  • Blue: Grammar structures and rules
  • Green: Examples and sample sentences
  • Purple: Communication phrases and expressions

File Formats

This repository contains materials in two formats:

  • Markdown (.md): Original source files that are easy to edit and maintain
  • HTML (.html): Web-friendly files with proper styling and color formatting

You can access either format based on your preference. The HTML files provide better visual formatting, especially for the color-coded content.
