Lektion 3 - Comic Strip Format

Frau Müllers Weg zur Gesundheit: Eine visuelle Geschichte

Diese Comic-Strip-Darstellung zeigt wichtige Szenen aus der Geschichte über Frau Müller und verwendet die Vokabeln und Grammatik aus Lektion 3. Die Beschreibungen der Bilder helfen Ihnen, sich die Szenen vorzustellen.

Farbkodierung (Color Coding):

  • Rot/Red: Vokabeln/Vocabulary
  • Blau/Blue: Passiv mit Modalverben/Passive with modal verbs
  • Grün/Green: Genitiv/Genitive case
  • Orange: Bruchzahlen/Fractions
  • Lila/Purple: Kommunikationsphrasen/Communication phrases

Szene 1: Stress bei der Arbeit

+-----------------------------------------------------------------------+
|                                                                       |
| [Bild: Frau Müller sitzt an ihrem Schreibtisch im Labor, umgeben von  |
| Dokumenten und Reagenzgläsern. Sie sieht erschöpft aus und hält sich  |
| den Magen.]                                                           |
|                                                                       |
| Gedankenblase: "Mein <span style="color:red;">**Magen** (stomach)</span> schmerzt schon wieder.     |
| Ich bin so <span style="color:red;">**erschöpft** (exhausted)</span>. <span style="color:red;">**Nachts** (at night)</span>   |
| kann ich kaum schlafen."                                              |
|                                                                       |
| Kollegin (im Hintergrund): "<span style="color:orange;">**Die Hälfte** (half)</span> unserer        |
| Abteilung leidet unter Stress!"                                       |
|                                                                       |

Anleitung zur Verwendung des Comic-Strips:

  1. Visualisieren Sie die Szenen: Lesen Sie die Beschreibungen und stellen Sie sich die Bilder vor. Dies hilft Ihnen, die Vokabeln und Grammatik in einem visuellen Kontext zu verankern.

  2. Üben Sie die Dialoge: Lesen Sie die Sprechblasen laut vor und achten Sie auf die korrekte Aussprache.

  3. Achten Sie auf die Farbkodierung: Die verschiedenen Farben helfen Ihnen, die grammatischen Strukturen und Vokabeln zu identifizieren.

  4. Erzählen Sie die Geschichte nach: Versuchen Sie, die Geschichte anhand der Comic-Panels nachzuerzählen, ohne auf den Text zu schauen.

  5. Erstellen Sie eigene Panels: Zeichnen Sie Ihre eigenen Comic-Panels zu weiteren Szenen der Geschichte und verwenden Sie dabei die Vokabeln und Grammatik aus Lektion 3.
